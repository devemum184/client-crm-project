# Автоматизация процесса работы с клиентами фирмы

## Описание проекта
Комплексное решение для автоматизации приема заявок на IT-услуги фирмы и мониторинга их выполнения. Состоит из Telegram-бота для клиентов и веб-дашборда для руководителей и менеджеров.

## Основной функционал
**Telegram-бот:**
- Вывод каталога предоставляемых услуг и цен.
- Оформление заявки в 2 клика.
- Просмотр статуса своих текущих заявок.

**Интерактивный дашборд:**
- Визуализация распределения заявок по статусам (в работе, новые, завершены).
- Таблица всех заявок клиентов в режиме реального времени.

## Запуск проекта
1. Клонируйте репозиторий: `git clone <ссылка>`
2. Установите зависимости: `pip install -r requirements.txt`
3. Вставьте свой токен бота в файл `config.py`.
4. Запустите комплекс: `python run.py`
5. Дашборд будет доступен по адресу: http://127.0.0.1:8050

## Технологии
- Python, python-telegram-bot, SQLite, Dash, Plotly
